Напишите метод с арументом типа String (строка). Подсчитайте количество строчных букв в этой строке (подсказка: воспользуйтесь методом chars класса String).

Имея список строк, напишите метод, который возвращает список всех строк, которые начинаются с буквы «а» (строчные буквы) и имеют ровно 3 буквы.

Пусть дан список строк List<String>. Найдите в нем строку, содержащую максимальное число строчных букв. Чтобы код правильно работал, когда входной список пуст, можете возвращать объект типа Optional<String>.
